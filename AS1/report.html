Report

My experience completing this project was a roller coaster full of ups and downs. One of my biggest challenges was making the hamburger icon responsive for the mobile view of the website. I struggled to make the button toggle the navigation on and off which ate up a lot of my time. The most enjoyable part of this project was the CSS and making all the design decisions for the portfolio.

I would say that I have enjoyed my module experience so far, despite the tough moments developing the portfolio website, I enjoyed attending Chris’s lectures as he made the three hours informational as well as entertaining.

At the beginning of the development process, the websites main colour was grey because I wanted the website to have simple colours, however towards the end of the development process I changed the colour to red to make the website more vibrant and colourful. For the fonts I just looked through fonts until I found one I liked (boring I know).

First, I started with the navigation for the website, using the nav tag and then created a hover effect so that when the cursor hovers over the navigation the text changes colour.

I then focused on the index page; I wanted the page to welcome the visitor as well as inform the user on what the portfolio consists of. Then I made the contact page using the label and input tags. I then made the project page using CSS classes to organise the placeholder images for the projects for the portfolio followed by the video demonstration page.
